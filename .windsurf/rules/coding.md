REGLAS DE GENERACIÓN DE CÓDIGO:
1. Si te pido una nueva página, DEBES incluir los scripts "loader" correspondientes (ej: header-loader.js).
2. NO escribas el código <nav> o <footer> dentro del HTML de la página. Usa un <div id="container-name"></div>.
3. Si necesito un elemento nuevo (ej. una tarjeta de producto), verifica si ya existe un estilo similar en `catalogo.html` antes de crear CSS nuevo.
