Privacidad

Política de Privacidad

En Fashion Box cuidamos tus datos con un enfoque práctico: recolectamos únicamente lo necesario, lo usamos para operar el servicio, y aplicamos medidas de seguridad para proteger tu información.

Última actualización: 17/12/2025

Recopilación de Datos

Qué información podemos almacenar para operar la plataforma.

  • Datos de identificación: nombre, apellidos, correo electrónico y teléfono (si aplica).
  • Autenticación: credenciales de acceso (contraseña almacenada como hash) o identificadores de terceros (por ejemplo, ID de Google si usas inicio de sesión con Google).
  • Datos de pedido: historial de compras, productos, cantidades, precios, facturación y estatus.
  • Datos logísticos: dirección de envío, referencias y datos necesarios para la entrega.
  • Datos técnicos: información básica del dispositivo/navegador y registros de actividad para seguridad.
No solicitamos información sensible innecesaria para la operación del servicio.

Uso de la Información

Para qué se utiliza la información recopilada.

  • Gestión de pedidos: creación, confirmación, cobro, facturación y seguimiento.
  • Logística de envíos: preparación, entrega, rastreo y atención post-venta.
  • Comunicación: notificaciones, confirmaciones, soporte y avisos operativos.
  • Prevención de fraude: validaciones de seguridad y control de accesos.
  • Mejora del servicio: análisis básico para corregir errores y optimizar la experiencia.

Seguridad

Medidas aplicadas para proteger tu información.

Contraseñas con hash
Las contraseñas se almacenan como hash (por ejemplo, en campos tipo passwordhash), no en texto plano.
Sesiones protegidas
Las sesiones se manejan con mecanismos de autenticación y tokens, evitando exponer credenciales.
Acceso restringido
Aplicamos controles de acceso y registro de eventos para reducir riesgos operativos.

Aunque aplicamos buenas prácticas, ningún sistema es 100% infalible. Si detectas un incidente, repórtalo.

Cookies

Cómo se usan para mantener tu sesión y preferencias.

  • Sesión: mantener autenticación activa y evitar que inicies sesión en cada página.
  • Preferencias: recordar configuraciones básicas (cuando aplique).
  • Seguridad: apoyar mecanismos antifraude y detección de accesos anómalos.

Puedes gestionar cookies desde la configuración de tu navegador. Algunas funciones podrían verse limitadas.
