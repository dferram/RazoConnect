Conciliación de Inventario

Filtros avanzados para auditoría y reconciliación de movimientos

¿Cómo usar esta página?

Objetivo: Conciliar (cuadrar) los movimientos de inventario contra documentos fuente.

Tipos de Movimientos:

  • Entradas de Almacén: Recepciones de Órdenes de Compra (productos que llegan de proveedores)
  • Sesiones de Auditoría: Conteos físicos de inventario (ajustes por diferencias encontradas)
  • Ajustes Manuales: Mermas, adiciones u otros ajustes registrados manualmente

Pasos:

  1. Seleccionar Tipo: Filtra por "Entrada de Almacén" (OC), "Sesión de Auditoría" o "Ajuste Manual"
  2. Filtrar por Fecha: Define el rango de fechas a analizar
  3. Revisar Tabla: Verifica cantidades, valores y referencias contra tu documento fuente
  4. Exportar PDF: Descarga el reporte con totales para conciliación física
